# 电力 EPC 技术标写作规范（元知识）

## 表述风格

- 以施工工艺、工序、质量控制指标为主线，采用连续技术叙述
- 避免宏观口号、空泛宣誓、IT 行业套话（如「赋能」「生态」「顶层设计」）
- 参数必须带单位；设备型号须与检索素材或全局工程信息一致
- 同一工程全篇电压等级、工期、规模不得前后矛盾

## 章节内容要点

| 章节类型 | 必须包含 |
|---------|---------|
| 工程概况 | 建设规模、电压等级、主要设备数量、地形条件、工程显著特点（只描述不写对策） |
| 项目目标 | 质量、工期、造价等概括性目标承诺；不写保证措施与工艺细节 |
| 施工组织 | 组织机构、劳动力、机具配置、平面布置原则 |
| 专项施工方案 | 工序流程、关键控制点、安全措施、质量标准 |
| 进度计划 | 与总工期一致，关键路径、里程碑节点 |
| 质量安全 | 引用电力行业规程，量化检验频次与合格标准 |

## 格式与引用

- 规范引用示例：GB 50168、DL/T 5168、国网/南网相关反事故措施（按项目实际）
- 关键参数标注格式：**[参数名] 数值+单位**
- 图表使用系统占位符，不手写 ASCII 图

## 篇幅原则

- 高分评分项对应章节应写足工艺细节与数据，不以空话凑字数
- 低分辅助项简明扼要，但仍须回应评分项原文
- 引用规范/标准号、品牌型号时，仅可使用招标文件、检索素材或全局事实中已出现的内容；缺依据处用 **[参数] 待核实**，禁止编造

## 常见空话（应避免）

- 「综上所述」「总体而言」「值得注意的是」「具有重要意义」等过渡/评价套话
- 「赋能」「顶层设计」「闭环管理」等互联网表述
- 连续多段以「本工程采用…」等相同句式开头
